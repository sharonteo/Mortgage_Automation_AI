# Mortgage_Automation_AI

Intelligent Document Processing, Classification, and Workflow Automation for Mortgage Operations
This project demonstrates how applied AI, LLM‑based automation, and multi‑agent workflows can streamline key steps in the mortgage process. It simulates a LoanDepot‑style system that ingests borrower documents, classifies them, extracts structured information, validates it against underwriting rules, and generates an underwriter‑ready summary.
The goal is to show how modern AI can reduce manual review, improve accuracy, and accelerate the mortgage experience for both customers and operations teams.
